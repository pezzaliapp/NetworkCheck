NetworkCheck

Capisci cosa è esposto nella tua rete e sul tuo computer.

Come funziona

Servizi esposti

macOS sudo lsof -nP -iTCP -sTCP:LISTEN
Windows netstat -ano -p TCP | findstr LISTENING
Linux sudo ss -tulpn
Comandi alternativi
macOS — mostra i tuoi indirizzi IP locali: ifconfig | grep "inet "
Linux — mostra i tuoi indirizzi IP locali: hostname -I
Linux — porte in ascolto con processo: ss -tulpn

Windows — dal PID al programma
